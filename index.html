Tables

Tables is an Android application that allows UCSD to meet other students over a meal. It's goal is to make the task of meeting new people in an unfamiliar environment easier, more efficient, and more enjoyable. It has verified account creation, profile creation, an intelligent matching algorithm to match those that share similar interests, and a built in chat room. In the future, I hope to add machine learning for the matching algorithm and yelp integration into the chat room to allow users to locate the best place to eat within the application.
