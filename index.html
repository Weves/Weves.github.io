Projects

Some projects I have been working on.

Chess AI

Based on DeepMind's AlphaZero paper, competing with friends on who can design, build, and train the best performing chess AI. Written with Python/TensorFlow and using a GTX 1080TI for training.

Hard Negative Mining for Correspondance

Working with Professor Manmohan Chandraker at UCSD, I am working on applying new hard negative mining techniques to existing state of the art correspondance networks. Specifically. I am modifying a PyTorch implementation of SuperPoint to use random sampling, GPU based KNN, and GPU based Locality Sensitive Hashing KNN to find hard negative examples during training time.

Review Sentiment Classification

As part of a NLP kaggle competition at UC San Diego, I built a word-embedding based, semi-supervised review sentiment classifier. I placed in the top 5% of all contestants. The point of the contest was creative feature design - we were restricted to using logistic regression for the final classification. My final design used regularized document embeddings (average of the word embeddings of the document) using FastText (trained with both the review datset and web crawls).

Tables

Tables is an Android application built by a team of students at UCSD led by myself. It allows UCSD to meet other students over a meal. It's goal is to make the task of meeting new people in an unfamiliar environment easier, more efficient, and more enjoyable. It has verified account creation, profile creation, an intelligent matching algorithm to match those that share similar interests, and a built in chat room. Tables was built in Android Studio with Java. We used Git/GitHub for version control and Firebase as the database to hold user information and for the chat rooms. The project also uses a layered design. Following MVC principles, classes are divided to deal with a specific set of goals. Additionally, many industry document were created throughout the development of the application, such as User Stories, Requirements, and a Database Schema (just to name a few).
