Projects

Some projects I have been working on.

Tables

Tables is an Android application built by myself as well as a team of other students that I led. It allows UCSD to meet other students over a meal. It's goal is to make the task of meeting new people in an unfamiliar environment easier, more efficient, and more enjoyable. It has verified account creation, profile creation, an intelligent matching algorithm to match those that share similar interests, and a built in chat room. In the future, I hope to add machine learning for the matching algorithm and yelp integration into the chat room to allow users to locate the best place to eat within the application.

Implementation

Tables was built in Android Studio with Java. We used Git/GitHub for version control and Firebase as the database to hold user information and for the chat rooms. The project also attempts a layered design. Following MVC principles, classes are divided to deal with a specific set of goals. Additionally, many industry document were created throughout the development of the application, such as User Stories, Requirements, and a Database Schema (just to name a few).

Blackjack Simulator

Blackjack Simulator is exactly what it sounds like: a program that simulates a blackjack game to predict the best move in a given situation. Given a hand, a number of players, a dealers face up card, and a deck state (that you can continuously update), it runs that situation many times for each possible move you could make (hit, stand, etc.) and calculates the expected return on each and reccomends the move that gives you the best average outcome.

Implementation

Blackjack simulator was built in Visual Studio using C# as a console application with no extra bells and whistles.
